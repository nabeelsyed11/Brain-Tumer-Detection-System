# 🧠 Brain Tumor Detection System (AI-Based)

An AI-powered web application that detects brain tumors from MRI scans using deep learning techniques. This system enables quick and efficient preliminary diagnosis through an easy-to-use interface.

🔗 **Live Demo:**  
https://brain-tumer-detection-system.streamlit.app/

---

## 🚀 Overview

The **Brain Tumor Detection System** is a deep learning-based application designed to analyze MRI brain images and predict the presence of tumors. It integrates artificial intelligence with a web interface to provide fast and accessible results.

---

## 🧩 Features

- 🧠 Detects brain tumors from MRI images  
- ⚡ Provides fast and real-time predictions  
- 🌐 Interactive and simple UI using Streamlit  
- 📊 Deep learning-based classification  
- 📁 Easy image upload functionality  
- 📍 Accessible online from anywhere  

---

## 🛠️ Tech Stack

**Frontend:**
- Streamlit  

**Backend / AI:**
- Python  
- TensorFlow / Keras  
- NumPy

**Deployment:**
- Streamlit Cloud  

---

## 🧪 How It Works

1. Upload an MRI brain scan image  
2. The system preprocesses the image (resize, normalize)  
3. The trained CNN model analyzes the image  
4. The system predicts:
   - Tumor Present  
   - No Tumor  
5. Result is displayed instantly

## ⚙️ Installation & Setup

Run this project locally:

```bash
# Clone the repository
git clone https://github.com/your-username/brain-tumor-detection.git

# Navigate to project directory
cd brain-tumor-detection

# Install required packages
pip install -r requirements.txt

# Run the application
streamlit run app.py
```
